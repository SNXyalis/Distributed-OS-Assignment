Εγχειρίδιο χρήσης

Όλες οι λειτουργίες που αποτελούν την ευθύνη του διαχειριστή παρέχονται στο Admin Menu .

Προσθήκη/Ανανέωση/Διαγραφή χρήστη

  1. Κατευθυνθείτε στο admin menu
  2. Επιλέξτε την επιλογή User
  3. Επιλέξτε την επιθυμητή λειτουργία
  4. Θα ανακατευθυνθείτε σε ένα παράθυρο για την συμπλήρωση των στοιχείων/id

Προσθήκη/Ανανέωση/Διαγραφή μεταπτυχιακού προγράμματος

  1. Κατευθυνθείτε στο admin menu
  2. Επιλέξτε την επιλογή Postgraduate Programs
  3. Επιλέξτε την επιθυμητή λειτουργία
  4. Θα ανακατευθυνθείτε σε ένα παράθυρο για την συμπλήρωση των στοιχείων/id
